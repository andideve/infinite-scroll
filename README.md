# infinite-scroll

A lightweight React library for automatically next page.

## Install

```bash
npm i @andideve/infinite-scroll
```

## Usage

```jsx
return (
  <InfiniteScroll hasMore={hasMore} loadFn={loadMore} threshold={1000}>
    {/* movie list */}
  </InfiniteScroll>
);
```
